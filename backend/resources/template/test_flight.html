The next steps are as follows. We know they're complicated, but that's Apple's fault. We're annoyed too.

  1. Follow this link to sign up for a Test Flight account
  2. Once you have entered your information and registered, Test Flight will send you an email
  3. From your iPhone, open the email, click the login link (from your iPhone!)
  4. On login, you should see, This device will need to be registered before it can access BETAs
  5. Click Register Device
  6. It will prompt you to install a profile: click the Install button
  7. You will return to the Register Device screen; the initial signup is complete

You will receive an email from Test Flight allowing you to download the Viewfinder app.
